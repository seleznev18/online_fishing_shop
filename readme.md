# Онлайн-магазин "Fishing-market"

Проект представляет собой интернет-магазин с каталогом товаров, корзиной и формой оформления заказа. Реализован полностью на HTML, CSS и JavaScript.

---

## Структура проекта

### `index.html`

Главная страница магазина. Содержит:

* каталог товаров;
* корзину с возможностью добавления, удаления и изменения количества товаров;
* форму оформления заказа;
* подключение CSS и JS файлов.

### `styles.css`

Основной файл стилей, задаёт визуальное оформление элементов страницы, шрифты, цвета и расположение блоков.

### `responsive.css`

Адаптивные стили для различных экранов.

### `cart.js`

Содержит логику работы корзины.

### `products.js`

Содержит массив объектов с данными товаров: название, цена, изображение, описание.

### `main.js`

Инициализация страницы и элементов приложения после загрузки DOM:

* рендеринг каталога товаров;
* установка обработчиков событий для корзины и модального окна.

---

## Реализованный функционал

* Отображение каталога товаров на странице;
* Добавление, удаление и изменение количества товаров в корзине;
* Автоматическое сохранение состояния корзины;
* Оформление заказа через форму;
* Адаптивный интерфейс для различных размеров экрана;
* Уведомления о добавлении товара в корзину.
